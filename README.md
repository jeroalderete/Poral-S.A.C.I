## Instalacion :warning:

### :running: Ejecutar Local

Clonar el proyecto

```sh
- git clone https://github.com/jeroalderete/Poral-S.A.C.I
```

Ir a la carpeta del proyecto

```bash
  cd Poral-S.A.C.I
```

Instalar dependencias

```bash
  npm install
```

Start server

```bash
  npm run dev
```
