## Instalacion :warning:

### :running: Ejecutar Local

Clonar el proyecto

```sh
- git clone https://github.com/jeroalderete/Batacazo-Ecommerce-Entrega-Final
```

Ir a la carpeta del proyecto

```bash
  cd Batacazo-Ecommerce-Entrega-Final
```

Instalar dependencias

```bash
  npm install
```

Start server

```bash
  npm run dev
```
